# GlobalEngine

![GlobalEngine](/resources/GlobalEngineLogo.png?raw=true "GlobalEngine")

This game engine is a pet project aimed to learn better C++ and OpenGL.

Make sure that you already pre installed packages:
- Make
- libglfw3
- libglfw3-dev
- libglew-dev
- libsoil-dev
- libgtest-dev
- libglm-dev
- Boost
